这次的决策顺序

  1. 到港后先去尖沙咀凯悦寄存行李，尽快让“人”和“行李”进入可控状态。
  2. 中银香港线上办卡放在酒店 WiFi 下完成，避免边走边传资料。
  3. 下午集中去尖沙咀 / 海港城一带找 GR III，必要时为现货牺牲一部分观光。
  4. D2 晚上只做尖沙咀夜景和晚餐，D3 再处理返程前的轻观光。

执行摘要

2 人情侣行 1 晚深圳 + 1 晚香港 任务型旅行 返程日最晚 16:00 到西九龙
  • D1 不安排跨区夜游，只做入住、吃饭、收资料。
  • D2 是真正的核心日，任务顺了，夜景自然顺。
  • D3 默认推荐“中环 + 山顶经典线”，旺角港味线作为更街头化的备选。

行程总览

顶部先看全局，不用翻到下面才知道重点。这里把酒店、任务、日期和打法全部压缩在一屏内。

D1 / 深圳

深圳只做轻量收口

3 月 19 日周四 20:30 到深圳北，酒店又在步行约 6 分钟范围内，最优解不是夜游，而是把第二天进港材料整理好，保证 D2 从一开始就流畅。

  • 晚餐只找酒店周边快吃，不跨区。
  • 睡前把通行证、车票、相机代购需求截图放一起。
D2 / 尖沙咀

尖沙咀凯悦是任务执行基地

这次香港不需要跑中银分行，所以凯悦的价值很高。寄存行李后就能在 WiFi 下办卡，再步行覆盖 K11、海港城、丰泽、百老汇、星光大道和海旁晚餐。

  • 先稳定网络和证件，再开始消费决策。
  • 下午优先买 GR III，观光往后放。
D3 / 返程前

D3 留给第一次来香港的经典感或街头感

3 月 21 日周六 17:30 从香港西九龙返程，D3 不是继续复制 D2 海旁夜景，而是集中给“第一次来香港”的半日观光窗口。你们可以在山顶经典线和旺角港味线之间二选一，但都必须中午后回尖沙咀拿行李。

  • 默认推荐中环 + 山顶经典线，记忆点最强。
  • 想更有小红书街头感，就选旺角 + 油麻地线。

先看怎么选

把这次旅行最容易纠结的三件事提前说清楚：什么时候做任务、什么时候逛、返程日前该不该贪心。

任务 1

中银香港借记卡必须放在酒店 WiFi 环境里做

这件事不再占用线下跑点时间，但仍然需要稳定网络、证件、地址信息和验证码配合。放在寄存完行李后做，错误率最低，也方便补资料。

任务 2

GR III 购机优先级高于一般观光

相机现货和门店配额都比夜景稀缺，所以 D2 下午要先把丰泽、百老汇、海港城一带走完。观光是锦上添花，购机才是这趟香港最容易失手的节点。

D3 决策

返程日前半天只在“经典首访”与“港味街头”之间二选一

D3 不再重复 D2 的尖沙咀海旁。默认推荐中环 + 山顶，把第一次来香港最该补的视角一次补齐；如果你们更想拍街头氛围和扫街，就切到旺角 + 星际城市 + 油麻地警署线。

三天时间轴

每一天都按“能不能执行”而不是“能不能塞满”来排。尤其 D2 和 D3，时间都是为任务和返程倒推出来的。

D1｜2026-03-19 周四

广州东 → 深圳北｜晚到、轻安排、不跨区

20:30 到深圳
  • 19:30

    广州东站出发，乘高铁前往深圳北站。

  • 20:30 - 20:45

    抵达深圳北后直接步行去酒店办理入住，不再额外打车换区。

  • 21:00 - 21:50

    酒店附近解决晚餐，优先粉面、粥铺、粤式快餐，核心是快吃快回房。

  • 22:00 - 22:30

    整理 D2 材料：通行证、车票、充电线、朋友代购截图、可用银行卡。

  • 22:30 后

    休息。第二天酒店到深圳北站很近，不用过早出门，但也不要故意拖慢。

  • 这晚不安排福田、南山、海上世界等夜游，收益太低。
  • 睡前确认手机电量和香港漫游 / 支付方式，能减少 D2 到港后的碎片焦虑。

D2｜2026-03-20 周五

深圳北 → 香港西九龙｜任务执行日 + 晚间轻观光

10:30 到港
  • 08:00 - 08:50

    起床、早餐、收拾行李，按退房节奏准备离店。

  • 09:15

    建议最晚到深圳北站。酒店离站近，步行进站即可。

  • 10:00 - 10:30

    深圳北站出发，10:30 抵达香港西九龙站。

  • 10:30 - 11:15

    通关后直接去尖沙咀凯悦寄存行李，不先逛街，不先吃大餐。

  • 11:20 - 12:00

    必做 1：回酒店大堂或休息区，用 WiFi 办理中银香港借记卡线上申请。

  • 12:05 - 13:00

    K11 / 河内道 / 金巴利道附近快速午餐，吃完直接进购机动线。

  • 13:10 - 15:30

    必做 2：集中购买 RICOH GR III。优先尖沙咀 / 海港城 / 广东道门店，尖沙咀无果时再把西九龙商圈作为备援。

  • 15:40 - 16:20

    回酒店正式入住，放好相机和资料，确认办卡资料是否提交完成。

  • 16:30 - 20:45

    进入收尾夜景线：凯悦出发，经 K11 或海港城解决晚餐，再走尖沙咀海旁、星光大道、1881 / 天星码头外围，最后直接回酒店。

  • 21:00 后

    视体力决定是否补一段海旁夜景，否则直接回凯悦休息。

  • D2 的关键不是玩多少，而是把“线上办卡 + 购机 + 入住”压缩到一个区域内完成。
  • 如果 GR III 现货难找，允许压缩夜景和晚餐时长，任务优先级不变。

D3｜2026-03-21 周六

香港半日游 → 香港西九龙 → 广州｜返程倒推日

17:30 返程
  • 08:20 - 09:10

  • 09:20 - 11:50

  • 12:00 - 13:10

    回尖沙咀午餐；山顶线建议在回程后吃正餐，旺角线则在佐敦 / 尖沙咀收口，不把午餐拖到更远区域。

  • 13:10 - 14:00

    回酒店取行李，检查证件、相机、线上办卡回执和充电设备。

  • 14:00 - 15:15

  • 15:20 - 15:40

    从尖沙咀凯悦出发前往香港西九龙站。

  • 返程日最怕把 D2 的维港夜景又走一遍，导致真正的新鲜感不够。
  • 只要 15:20 前还没回到酒店附近，就说明路线已经过长，需要立刻收缩。

办卡与购机清单

这一页不是让你现场才想起准备什么。下面两张卡片就是 D2 的任务脚本，出门前就可以照着勾。

中银香港借记卡

酒店 WiFi 下办理，避免边走边传资料

  • 优先准备：通行证 / 身份证明、过关凭据、住址信息、常用手机号、税务居民信息。
  • 如果 BOCHK App 或网页要求补件，直接在酒店环境里处理，不要站在商场门口临时填表。
  • 官方开立账户帮助页显示，单名账户线上开立储蓄 / 往来账户服务时间一般为 08:00 - 22:00；你们 D2 中午办理在时间窗内。
  • 以 BOCHK App 实时资格校验为准。页面里只把“线上办理”作为执行建议，不强行承诺当场必过。
RICOH GR III 代购

先问现货，再成交，再验机

  • 优先顺序：尖沙咀 / 海港城 / 丰泽 / 百老汇等近距离门店，先打电话或到店确认有无 GR III 现货。
  • 2026-03-07 检索到 Fortress 公开商品页仍列有 GR III / GR III HDF 商品信息，但门店现货和展示价可能波动，实际以当日库存为准。
  • 验机重点：包装封条、机身序列号、镜头伸缩、对焦、触屏、Wi-Fi、热靴、充电线与保修信息。
  • 买完先回酒店放好，不建议带着新相机在晚高峰购物区继续久逛。

路线选择

D2 尖沙咀任务收尾夜景线

凯悦 -> K11 或海港城晚餐段 -> 尖沙咀海旁 -> 星光大道 -> 1881 / 天星码头外围 -> 回酒店。

  • 适合时段：2026-03-20 16:30-21:00。
  • 跨海：否。情侣友好度高，最适合完成任务后的放松收尾。
  • 用餐建议：午餐快吃，晚餐放在 K11 或海港城，甜品可视体力删减。

D3 默认：中环 + 山顶经典首访线

凯悦 -> 中环 -> Peak Tram Lower Terminus -> 山顶 / 凌霄阁 -> 回尖沙咀午餐 -> 酒店取行李 -> 西九龙。

  • 适合第一次来香港，优先补齐港岛天际线和山顶视角。
  • 必须早出发，不扩展到半山、大馆或上环。
  • 山顶只做轻食或饮品，正餐放回尖沙咀。

D3 备选：旺角 + 星际城市 + 油麻地警署线

凯悦 -> 旺角 -> 西洋菜南街 / 女人街外围 -> 星际城市 -> 油麻地警署 -> 佐敦 / 庙街午餐 -> 回酒店取行李 -> 西九龙。

  • 更偏小红书港味街头感，不跨海，不重复维港夜景。
  • 适合喜欢扫街、拍路牌和霓虹街景的情侣。
  • 不再追加铜锣湾或尖沙咀海旁，保持返程节奏。

办卡与购机提醒

中银香港线上办理

  • 准备通行证 / 身份证明、过关凭据、住址信息、常用手机号、税务居民信息。
  • 优先在凯悦 WiFi 下完成，方便补件和验证码处理。

  • 优先尖沙咀与海港城片区，必要时再扩到西九龙。
  • 先问库存再移动，先确认现货再成交，买完先回酒店放置。
  • 验机重点：封条、序列号、镜头伸缩、对焦、触屏、Wi-Fi、配件与保修信息。
