D1 深圳收口

只做入住、晚餐、第二天资料整理，不安排跨区夜游。目标是把精神状态留给 D2 任务日。

D2 尖沙咀任务日

先寄存行李，再在酒店 WiFi 下处理 BOCHK 线上办理；下午集中完成 GR III 搜货与成交，晚上只走海旁收尾线。

D3 纯观光日

上午只允许二选一：第一次来香港默认走中环 + 山顶；若更想拍港味街头，再切换旺角 + 油麻地线。

返程硬锁

12:00 - 13:10 左右回尖沙咀午餐，13:10 - 14:00 回酒店拿行李，15:20 - 15:40 出发，16:00 前后到站。

D1 · 2026-03-19 周四

深圳轻量收口夜

不跨区 不熬夜 为 D2 留体力
19:30 - 20:30 广州东 → 深圳北

固定车次进深圳

这段不再加任何景点判断，按到站后尽快进入“入住 - 吃饭 - 收资料”的节奏。

20:30 - 21:20 出站 + 步行到酒店

先入住，压掉当天不确定性

酒店就在深圳北站附近，步行约 6 分钟，适合晚到后快速安顿，不值得再跨区折腾。

21:20 - 22:10 站边晚餐

选快吃型晚餐

目标不是“打卡”，而是补能量。优先热食、上菜稳定、离酒店近的店，避免需要排队的商场餐饮。

22:10 - 23:00 回房整理

把 D2 材料提前清好

  • 检查港澳通行证、过关票据、酒店订单、手机流量和支付方式。
  • 提前下载 BOCHK App，并把身份资料、地址、手机号、邮箱、TIN 等信息放到同一备注页。
  • 整理 GR III 验机清单，第二天到港后不临时慌乱。
D2 · 2026-03-20 周五

任务日 + 九龙海旁夜景收尾

寄存行李优先 酒店 WiFi 办卡 GR III 下午优先
09:15 - 10:30 深圳北 → 香港西九龙

10:00 发车，10:30 到港

到港后不先去逛，直接执行“寄存 - 办卡 - 购机”。任务没有完成前，不给自己加游玩支线。

10:30 - 11:20 西九龙 → 尖沙咀凯悦

先去酒店寄存行李

人先轻下来，下午搜货才不会被箱包拖累。尖沙咀凯悦本身就在后续主战场边上，能把动线压得很紧。

11:20 - 12:20 酒店 WiFi / 大堂休息区

处理中银香港借记卡线上办理

BOCHK 线上开户已经替代线下跑分行。酒店 WiFi 更适合上传资料、做人脸识别和接收短信，失败也方便立刻补资料。

  • 先核验 eligibility，再进入开户流程，不预设“必然成功”。
  • 若遇到资格不符、影像识别失败、补件或人工审核，立即止损，不让它吞掉下午购机时间。
12:20 - 13:20 尖沙咀快午餐

午餐要短、稳、离主战场近

优先海港城 / 广东道一带快进快出的商场餐饮，目标是给下午空出完整的搜货窗口，而不是把中午吃成一次单独行程。

13:20 - 16:50 尖沙咀 / 海港城 / 广东道

GR III 购机主战场

  • 第一轮优先打电话或线上问库存，再决定先去哪家门店，避免在广东道来回白跑。
  • 主搜索范围锁定尖沙咀、海港城、广东道；若主战场无果，再把西九龙作为同日备援。
  • 未确认现货前不付款；确认货号、颜色、是否港行、能否现场开箱验机后再成交。
16:50 - 17:30 回酒店短休 / 放机身

买到就先放回酒店

GR III 优先级高于景点，成交后第一动作是回酒店放好，不带着新机继续高强度逛街。

18:10 - 21:10 奖励式夜景收尾线

凯悦 → K11 / 海港城晚餐段 → 海旁 → 星光大道 → 1881 / 天星码头外围 → 回酒店

这一段只服务于“任务完成后的收口”，不再跳去旺角、油麻地、中环，也不做第二条夜游备选线。它是奖励，不是 D3 观光主线。

D3 · 2026-03-21 周六

纯观光日，只能二选一

上午出发 中午回尖沙咀 下午只做返程
默认推荐：中环 + 山顶经典首访线 备选：旺角 + 星际城市 + 油麻地警署港味线
08:00 - 08:40 早餐 + 出发

上午必须早走

不论选哪条线，上午就是唯一能自由使用的观光窗口；如果磨到中午才出门，整天的观光价值会急剧下降。

08:40 - 11:50 只走一条主线

默认推荐走中环 + 山顶；若更想拍港味街头，再切旺角线

两条线互斥，不跨海叠加，也不再塞回尖沙咀海旁。把“第一次来香港的经典感”和“港风街头感”清晰分开，减少纠结。

12:00 - 13:10 回尖沙咀午餐

午餐是返程前的节奏缓冲

不要把午餐安排在远处，更不要中午后还在新景点继续加点。此时应该进入回收动线，不再增加不确定性。

13:10 - 14:00 酒店取行李

必须先回酒店拿行李

返程约束高于所有观光。拿到行李后，下午就不应再安排任何远点位和排队项目。

15:20 - 15:40 酒店出发去西九龙

控制出发窗口

把这段当作硬门槛，而不是“参考时间”。宁可在站内多等，也不要赌周六下午尖沙咀到西九龙的路上不出波动。

16:00 前后 到达西九龙

17:30 固定返广

到站后只做安检、候车、简单补给。D3 中午以后不再继续观光，是整个三天里最不能放松的规则。

D3 的正确心法不是“还能不能再塞一个点”，而是“怎么确保观光完整又不误车”。只要过了 12:00，所有决策都必须服从回尖沙咀、拿行李、进西九龙。

D2 固定主线

尖沙咀任务收尾夜景线

适用日期：D2 晚上 是否跨海：否 赶时间友好：高 情侣适配：高

这是任务完成后的收口线，不是第二天主观光。把 K11 / 海港城晚餐、尖沙咀海旁、星光大道与天星码头外围串成一条紧凑的奖励路线。

尖沙咀凯悦

从酒店直接开始，节省转场成本；如果下午购机顺利，先回房放下战利品再出门最稳。

K11 / 海港城晚餐段

适合选一顿不需要排太久的商场晚餐。吃饭和海旁之间衔接顺，不会把节奏拖慢。

尖沙咀海旁 + 星光大道

重点是维港夜景和散步感。D2 已经看过海旁后，D3 就不再把维港当主体验，避免重复。

1881 / 天星码头外围

作为最后收尾点更合理，灯光氛围足，但又不把夜里动线拉得太远。

沿路吃什么

晚餐选商场内稳定出餐的港式 / 日式 / 茶餐厅最省时；海旁段更适合买饮品或小甜品边走边看景。D2 不建议排队吃网红大店，因为它会直接挤压任务缓冲时间。

明确排除：旺角、油麻地、中环都不在这晚出现。它的价值就是“近、顺、漂亮、好收口”。

D3 默认推荐

中环 + 山顶经典首访线

适用日期：D3 上午 是否跨海：是 赶时间友好：中高 情侣适配：很高

如果这是第一次来香港，这条线更值得优先给。它提供的是“经典香港”的俯瞰视角，而不是 D2 已经看过的海旁夜景重复版。

凯悦 → 过海去中环

要早出发，才能把交通、排队和观景都留在上午完成，不挤压返程前的回收窗口。

Peak Tram Lower Terminus

山顶缆车本身就是经典体验，适合第一次来香港时优先打卡。官方建议早去更利于避开排队高峰。

山顶 / 凌霄阁

价值在于港岛与维港整体天际线的“开阔感”。它和 D2 的海旁散步是两种完全不同的观看方式。

回尖沙咀午餐 → 酒店取行李

只保留主体验，不额外延展去大馆、半山扶梯、上环，确保中午前能稳妥回收。

沿路吃什么

早餐宜在出发前快吃，别把第一餐变成排队项目；午餐放回尖沙咀最合理，既能休整，也能和取行李动作拼在一起。若想边看景边吃，把轻食和咖啡留给山顶，不建议在中环临时扩线找店。

这是“第一次来香港更值得去”的主线。核心是山顶经典感，不再加大馆、半山扶梯、上环等支线。

D3 备选路线

旺角 + 星际城市 + 油麻地警署港味线

适用日期：D3 上午 是否跨海：否 赶时间友好：高 情侣适配：高

这条线更偏街头、扫街和小红书常见的港味氛围感。重点不是经典俯瞰，而是招牌、商场、街市和旧九龙的密集节奏。

凯悦 → 旺角

全程不跨海，移动成本低，适合作为“更稳的返程日上午线”。

西洋菜南街 / 女人街外围

女人街和旺角街区适合拍人流、招牌和摊位密度，氛围感更强，典型港风街拍素材集中。

星际城市

补一段经典港式商场感，让这条线不只剩街拍，也多一点老派电子与模型迷的气质。

油麻地警署 → 佐敦 / 庙街午餐

油麻地警署属于高辨识度港剧地标；庙街一带更适合把“街头氛围”收成一顿午餐，再顺势回酒店。

沿路吃什么

早餐可在尖沙咀快吃后上路；旺角段适合奶茶、面包、烧味等短停补给；真正坐下来吃饭，建议放在佐敦 / 庙街附近，方便控制回酒店节奏。返程日前不要把街头线吃成多段排队打卡。

铜锣湾不作为独立主线；如果你们比起街头拍照更想购物，可以在文案层面把它当作“替换旺角线”的次级说明，但不再扩成第四条路线。

办卡与购机清单

把风险点写清楚，避免现场靠记忆做判断。

BOCHK 借记卡：线上办理提醒

地点：尖沙咀凯悦酒店 WiFi 方式：App 线上办理 结果：以资格校验与官方审核为准
  • 先确认自己是否符合 BOCHK Mobile Account Opening 的资格边界，再开始流程。
  • 准备有效身份证明、手机号、住址、邮箱、TIN 等常见资料；若使用 `iAM Smart` 或拍卡验证，按官方流程执行。
  • 官方页面说明，18 岁及以上用户在周一至周六 09:00 - 20:00（不含公众假期）内成功开户，账户可较快进入可用状态；若在非服务时段提交，会延到下一工作时段处理。
  • 页面不承诺必开成功，也不建议为此专门跑分行。若流程卡住，优先保住 D2 下午购机时间。

RICOH GR III：代购执行策略

优先级：高于普通观光 主战场：尖沙咀 / 海港城 / 广东道 备援：西九龙
  • 先电话或在线问库存，再移动；不要靠“到店碰碰运气”浪费广东道来回时间。
  • 先确认是 GR III 还是 GR III HDF / GR IIIx，避免朋友要的是标准款却被现场节奏带跑。
  • 成交前核对：港行版本、机身序列号、保修方式、发票抬头、包装封条、配件是否齐全、电池循环状态是否正常。
  • 现场验机至少确认：开关机、镜头伸缩、对焦、触屏、拨轮、热靴、闪光同步、WiFi / 蓝牙、像素坏点、机身外观与镜头无明显灰尘或划伤。
  • 买完先回酒店放置，再去吃饭和走海旁夜景线，不背着新相机继续长距离移动。
